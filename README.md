# Treasure_Hunter
Lien vers le jeu : https://sunlyimo.github.io/Treasure_Hunter/

Ceci est un petit projet individuel en développement web sur Javascript.

Le but du jeu est de trouver le trésor grâce à des indications à chaque tour, avec 15 tours maximum.

J'aime bien ce jeu, j'ai ajouté des émojis pour un peu plus d'interactivité avec le joueur.
